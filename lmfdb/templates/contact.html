Contact LMFDB developers

For questions, inquiries, feedback or any other issues, including bug reports, please email the LMFDB support mailing list.

If you see a problem with the production site www.lmfdb.org, before making a report, please first try with the same URL except with 'www' replaced by 'beta', that is at beta.lmfdb.org, which runs our beta testing website code, since the issue you see may have already been addressed.

To be most helpful, bug reports should include

  1. a one line description of the problem
  2. the exact URL which shows the issue (copied and pasted from the browser location bar)
  3. a full description of the problem.

The current list of issues being tracked can be found here.

Use of the old Feedback Form is deprecated.
