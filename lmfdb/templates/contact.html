Contacting the LMFDB developers

For questions, inquiries, feedback or any other issues, including bug reports, please either email the LMFDB support mailing list, or use the Feedback Form, or both.

If you see a problem with the website www.lmfdb.org, before making a report, please check to see if the issue has already been fixed on the beta testing site beta.lmfdb.org: just replace 'www' replaced by 'beta' in the URL.

To be most helpful, bug reports should include

  • a one line description of the problem,
  • the exact URL which shows the issue (copied and pasted from the browser location bar), and
  • a full description of the problem.

The current list of issues being tracked can be found here.
