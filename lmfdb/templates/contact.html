Contacting the LMFDB developers

If you are interested in becoming involved with the LMFDB, please email the LMFDB mailing list.

To report an error or ask a question about the website content, please either email the LMFDB support mailing list, or use the Feedback Form, or both.

To be most helpful, error reports should include

  • a one line description of the problem,
  • the exact URL which shows the issue (copied and pasted from the browser location bar), and
  • a full description of the problem.

The current list of issues being tracked can be found here.
