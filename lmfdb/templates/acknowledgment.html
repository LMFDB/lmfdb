Software

Our website is run on free open source software. We use Sage for computations on the fly. Some important components used are
  • PARI
  • eclib
  • lcalc
The non-mathematical software used includes
  • Python
  • Flask
  • Jinja
  • MongoDB
  • pymongo
  • Mercurial
  • Git
Some of the data on this website was computed using other programs, including
  • Magma
  • Mathematica
