Software

Our website is run on free open source software. We use SageMath for computations on the fly. Some important components used are PARI, eclib and lcalc. The non-mathematical software used includes Python, Flask, Jinja, MongoDB, pymongo, and Git. Some of the data on this website was computed using other programs, including Magma and Mathematica.

Hosting

The database and web server were hosted until September 2013 on William Stein's NSF-funded cluster at the University of Washington. The database and web server have been hosted since September 2013 on the EPSRC-funded cluster at the University of Warwick. The website source code is hosted at GitHub.
